A comprehensive task management solution designed for small teams. Key features include:

- **Kanban Board** — Drag-and-drop task organization across customizable columns
- **Real-time Collaboration** — WebSocket-powered live updates when team members make changes
- **Task Dependencies** — Define task relationships and see critical path analysis
- **Filters & Search** — Advanced filtering by assignee, priority, label, and due date
- **API** — RESTful API with comprehensive documentation for integrations
