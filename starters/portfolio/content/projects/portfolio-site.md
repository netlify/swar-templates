This portfolio site showcases my work and experience using modern web technologies. Built with TanStack Start for full-stack React with file-based routing, it features:

- **AI Resume Assistant** — An interactive chatbot that answers questions about my experience
- **Content Collections** — Markdown-driven content for jobs, education, blog posts, and projects
- **Responsive Design** — Fully responsive layout with mobile navigation drawer
- **Blog** — Technical blog posts with syntax highlighting and tag filtering
- **Contact Form** — Netlify Forms integration for easy contact
